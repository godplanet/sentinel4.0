NAVIGATION PATTERNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TYPE A (Direct Link):
• No chevron icon
• Single click to navigate
• Example: KOKPİT → /dashboard

TYPE B (Expandable Group):
• Chevron icon (▼/▶)
• Click to expand/collapse
• Children indent under header
• Example: STRATEJİ & YÖNETİŞİM ▼
           └─ Denetim Evreni

ACTIVE STATE:
• White background highlight
• Ring border (ring-1 ring-white/10)
• Bold text
• Example: Currently viewing /execution/findings

BADGES:
• Color-coded status indicators
• Positioned right-aligned in item
• Support on both L1 and L2 levels
• Example: Bulgu Merkezi [HOT] (red)

TABS:
• Shown in PageHeader component
• Horizontal navigation within page
• Active tab underlined (blue)
• Example: Dashboard → [Genel | Stratejik | Ekosistem]
